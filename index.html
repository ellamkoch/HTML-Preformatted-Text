Exploring Pre-formatted Text in HTML

<pre> is an element/tag that preseves spaces, tabs and line breaks instead of ignoring them. It's used to display text exactly as it is written/typed in the HTML file. Examples of use include:

  • Displaying code snippets
  • Showing ASCII art
  • Presenting formatted text like poetry or tabular data
